# boilerplate-cli
> boilerplate-cli-description

## usage
| api  | params         | description            |
| ---- | -------------- | ---------------------- |
| say  | prefix/wow(-w) | Say what u want to say |
| jump | distance(-d)   | Jump to where          |

```shell
boilerplate-cli say what-hello -w=hello,fei
boilerplate-cli jump -d=123
```
