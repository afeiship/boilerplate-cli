## installation
```shell
npm install @feizheng/boilerplate-cli
```

## usage
~~~
Usage: boilerplate-cli [options]

Options:
  -V, --version  output the version number
  -d, --debug    only show cmds, but not clean.
  -h, --help     display help for command
~~~
